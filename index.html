01 / Sobre mí

¿Quién
soy?

Soy Andrés, un desarrollador a punto de completar el ciclo formativo de DAW. Me especializo en el desarrollo full-stack, con experiencia en backend con PHP, Java y C#, y en frontend con HTML, CSS, JavaScript y Vue.js.

Mi motivación es entender cómo funcionan las cosas por dentro: desde la arquitectura de un servidor hasta la experiencia final del usuario. Complemento mi formación con proyectos propios en los que aplico tecnologías como Docker y AWS para trabajar en entornos lo más cercanos posible al mundo real.

02 / Habilidades

Mi caja de herramientas

Backend

PHP

Desarrollo de APIs REST y lógica de servidor. Uso servidores Apache en AWS para desplegar proyectos propios, incluyendo un servidor de Minecraft.

Java · Spring Boot

Desarrollo de servicios backend con Spring Boot: controladores REST, inyección de dependencias y gestión de entidades con JPA.

C# · Unity · ASP.NET

Programación de mecánicas en Unity para videojuegos e iniciación al desarrollo web con ASP.NET Core para aplicaciones empresariales.

Frontend

HTML5 / CSS3

Maquetación de interfaces web semánticas y accesibles. Diseño responsive con Flexbox y Grid para adaptarlas a cualquier dispositivo.

JavaScript

Lógica del lado del cliente, manipulación del DOM y consumo de APIs REST. Base de la interactividad en todos mis proyectos web.

Vue.js

Construcción de SPAs con componentes reutilizables y reactividad declarativa. Gestión del estado de la interfaz en aplicaciones de página única.

DevOps & Cloud

Docker

Contenerización de aplicaciones para garantizar entornos reproducibles. Despliegue de servicios aislados y portables en distintos entornos de ejecución.

AWS

Administración de instancias EC2, almacenamiento con S3 y primeros pasos con funciones Lambda. Infraestructura en la nube para proyectos propios.

Bases de Datos

MongoDB

Modelado de datos no relacionales con documentos JSON. Utilizado en proyectos donde la estructura de datos es flexible y variable.

MySQL

Diseño de esquemas relacionales, consultas optimizadas y gestión de datos en aplicaciones PHP y Java con relaciones entre entidades.

SQLite

Almacenamiento local en aplicaciones de escritorio y prototipos. Solución ligera sin necesidad de servidor para persistencia de datos local.
